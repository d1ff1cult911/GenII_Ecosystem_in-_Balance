# GenII_Ecosystem_in-_Balance
Экосистема в балансе
Образовательный симулятор экосистемы для WebSim

Описание
"Экосистема в балансе" — это стратегический симулятор, в котором игрок выступает в роли эколога и управляет разнообразной экосистемой. Задача — поддерживать баланс между растениями, травоядными и хищниками в условиях динамично меняющихся климата и случайных природных событий. Игровая механика построена на косвенном управлении экосистемой через изменение параметров среды и мудрое распределение ресурсов.

Основные функции
Косвенное управление экосистемой: влияние на климат, влажность, пополнение видов

Моделирование цепочек питания, динамичные популяции и их взаимодействия

Система случайных событий: катастрофы, болезни, инвазивные виды

Графическая визуализация баланса, здоровья и динамики всех видов

Условия победы и поражения ориентируются на долгосрочную устойчивость
